1. Your Client ID

No Client Secret needed — the extension uses PKCE (no secret stored).

2. Redirect URI (add this in your app)

In your Spotify app settings, open Redirect URIs and add this exact URL:

The full URL usually ends with .chromiumapp.org/. Use Copy to get the exact value, then in Spotify Developer Dashboard → Your app → Settings → Redirect URIs → Add, paste it and save.

If you get "Authorization page could not be loaded": copy the URL above again and in Spotify remove the old extension redirect URI, then Add this one (exact match required). Reloading the extension can change the URL.

3. Quick setup

  1. Go to Spotify Developer Dashboard and log in.
  2. Create an app (or use an existing one).
  3. Open your app → Settings → under Redirect URIs, click Add and paste the URL from step 2 above.
  4. Save. Then copy your Client ID from the app overview and paste it in the field in section 1.
  5. Click Save below.
